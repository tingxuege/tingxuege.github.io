weiming-health接口文档

版本 创建人 时间
V1.0 程泉 2018-05-15

1.获取服务列表接口

  • 下拉框
  1. 接口地址：/view/application
  2. 支持格式：json
  3. 请求方式：get
  4. 请求参数
名称 必填 类型 说明
application 否 String 默认显示的服务名称
不填则返回第一条

5.返回字段说明

名称 说明
application 默认展示的服务名称
data 服务列表

JSON返回示例

2.获取图表信息接口

  • Echarts图表所需数据
  • 仪表盘展示数据
  1. 接口地址：/view/detail
  2. 支持格式：json
  3. 请求方式：get
  4. 请求参数
名称 必填 类型 说明
application 是 String 选取需要展示的服务名称
time 是 int 图标展示的时间段

5.返回字段说明

  • 整体信息
名称 说明
application 服务名称
balance 平均负载
runnable 线程信息（折线图）
tomcat tomcat容器的会话使用情况（折线图）
heap 堆内存使用情况（折线图）
noheap 非堆内存使用情况（折线图）
gc 垃圾回收信息（折线图）
clazz 应用类信息（折线图）
name ip地址
cpu cpu数量
id 应用特征码
status 服务状态
  • Echarts所需数据
名称 说明
stack y轴名称
data y轴数据
name 具体折线名称
type 图表类型（默认折线图）
line 折线名称列表
time x轴数据

JSON返回示例
